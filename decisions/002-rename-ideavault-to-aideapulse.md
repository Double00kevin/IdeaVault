# ADR-002: Rename IdeaVault → AIdeaPulse

**Date:** 2026-03-27
**Status:** Accepted

## Context

"IdeaVault" was the working name. All domain extensions (.com, .io, .ai, .app) were taken. Needed a name that:
- Communicates AI-powered idea discovery
- Sounds like a modern SaaS product
- Has a clean .com available
- Works well with hero copy and marketing

## Decision

Rename to **AIdeaPulse** (`aideapulse.com`).

- "AI" is embedded in the word "Aidea" (AI + Idea)
- "Pulse" conveys real-time, always-on signal scanning
- Domain registered on Cloudflare Registrar (2026-03-27)
- Undercuts competitor IdeaBrowser ($299-999/yr) at $12/mo

## Consequences

- Pro: Strong, unique name with .com available
- Pro: AI positioning baked into the brand
- Pro: Domain on same platform as Workers/Pages/D1 (Cloudflare)
- Con: Requires rename across codebase, Workers, D1, Stripe, Clerk, and all docs
- Con: GitHub repo rename needed
